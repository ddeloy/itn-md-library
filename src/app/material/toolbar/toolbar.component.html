Toolbar md-toolbar is a horizontally aligned bar, which can hold the application title and/or actions.

MdToolbarModule - Official Documentation

A toolbar is created by using md-toolbar element.

Below example creates a toolbar with a custom nested menu in it. Please see Menu page for details on Custom Nested Menu.

Default Toolbar 
<md-toolbar>
  // this button code is for collapsing the menu to 3-lines btn on mobile screen
  <button md-button md-icon-button class="button-collapse" (click)="onMenuClick()">
    <md-icon>menu</md-icon>
  </button>
  <a href="#" class]="brand-logo">Logo</a>
  <top-menu [items]="menuItems" [ngClass]="{{'{'}}'shown': showMenu}"></top-menu>

Toolbar Theming

By default, toolbars will use light greyish shade (rgba(243, 243, 243, 0.41)) as their background color. They can, however, be themed by setting color attribute to primary, accent or warn.

Basic Menu 
<md-toolbar color="primary">
  <span>Primary Toolbar</span>
  <span fxFlex></span>
  <button md-button md-icon-button>
    <md-icon>more_vert</md-icon>
  </button>
</md-toolbar>
