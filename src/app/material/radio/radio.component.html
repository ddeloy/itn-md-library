Radio Button A material design radio button that supports all of the functionality of native radio button element

MdRadioModule - Official Documentation

Radio-button element is created using md-radio-button element. Radio-button label is provided as the content to this md-radio-button element.

Radio-button - Basic Example 
<md-radio-button name="group1">Option 1</md-radio-button>
<md-radio-button name="group1">Option 2</md-radio-button>
<md-radio-button name="group1">Option 3</md-radio-button>
<md-radio-button name="group1" disabled>Option 4 (disabled)</md-radio-button>
Option 1 Option 2 Option 3 Option 4 (disabled)

Theming

By default, radio-buttons use theme's accent color. This can be changed by setting color property on md-radio-button element. Allowed values are primary, accent and warn.

NOTE: According to website design, radio-buttons should use primary (green) color. Therefore, make sure to set color property to primary on the md-radio-button element.

Radio-button Theming 
<md-radio-button color="primary" name="group2">Primary</md-radio-button>
<md-radio-button color="accent" name="group2">Accent/Secondary</md-radio-button>
<md-radio-button color="warn" name="group2">Warn</md-radio-button>
Primary Accent/Secondary Warn

Radio Groups

Related radio-buttons should typically be placed inside of an md-radio-group element. All the md-radio-button elements inside a md-radio-group element will inherit the name property of the group. And disabling the group will disable all the radio-buttons contained in it. Also, md-radio-group has a value property that reflects the currently selected radio-button inside of the group.

Radio Group 
<md-radio-group name="my_options" [disabled]="isDisabled">
  <md-radio-button color="primary" value="option_1">Option 1</md-radio-button>
  <md-radio-button color="primary" value="option_2">Option 2</md-radio-button>
  <md-radio-button color="primary" value="option_3">Option 3</md-radio-button>
</md-radio-group>
<button md-raised-button md-button-sm color="primary" (click)="isDisabled=!isDisabled">
  {{'{'}}{{'{'}}isDisabled ? 'Enable' : 'Disable'}}
</button>

Alignment

Radio-button can be positioned before or after its label by setting align attribute to start or end, respectively. If align is not set, then radio-button would be aligned before its label.

Radio-buttons inherit align property of their parent md-radio-group element.

Radio-button Alignment 
<md-radio-group fxLayout="column" name="my_options1" [align]="isAlignEnd ? 'end' : 'start'">
  <md-radio-button color="primary" value="option_1">Option 1</md-radio-button>
  <md-radio-button color="primary" value="option_2">Option 2</md-radio-button>
  <md-radio-button color="primary" value="option_3">Option 3</md-radio-button>
</md-radio-group>
<md-checkbox color="primary" [(ngModel)]="isAlignEnd">Align end</md-checkbox>
