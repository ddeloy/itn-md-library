Checkbox A material design checkbox component that supports all of the functionality of an HTML5 checkbox

MdCheckboxModule - Official Documentation

Checkbox element is created using md-checkbox element. Checkbox label is provided as the content to this md-checkbox element.

Checkbox - Basic Example 
<md-checkbox>
  Enabled Checkbox
</md-checkbox>
<md-checkbox indeterminate="true">
  Indeterminate Checkbox
</md-checkbox>
<md-checkbox disabled>
  Disabled Checkbox
</md-checkbox>
Enabled Checkbox Indeterminate Checkbox Disabled Checkbox

Theming

By default, checkboxes use theme's accent color. This can be changed by setting color property on md-checkbox element. Allowed values are primary, accent and warn.

NOTE: According to website design, checkboxes should use primary (green) color. Therefore, make sure to set color property to primary on the md-checkbox element.

Checkbox Theming 
<md-checkbox color="primary">
  Primary Checkbox
</md-checkbox>
<md-checkbox color="accent">
  Accent Checkbox
</md-checkbox>
<md-checkbox color="warn">
  Warn Checkbox
</md-checkbox>
Primary Checkbox Accent Checkbox Warn Checkbox

Alignment

Checkbox can be positioned before or after its label by setting align attribute to start or end, respectively. If align is not set, then checkbox would be aligned before its label.

Checkbox Alignment 
<md-checkbox color="primary" align="start">
  Before Label
</md-checkbox>
<md-checkbox color="primary" align="end">
  After Label
</md-checkbox>
