Chips An input component for building lists of strings or objects. The list items are displayed as 'chips'.

MdChipsModule - Official Documentation

Chips are created by using md-chip elements, and a list of such chips is created by wrapping these md-chip elements inside a md-chip-list element.

Modification

Chips have been modified to include one close (X) button on the right hand side. Clicking on this button should delete that chip.

NOTE: onClick function for this close button needs to be implemented as per requirement.

Selectable Chips

Chips can be selected by setting selected attribute on md-chip element to true. Selection can be disabled by setting selectable to false on the md-chip-list.

Color for selected chips can be set by setting color attribute on md-chip to primary, accent or warn. If color attribute is not set then primary color is used.

Selectable Chips 
<md-chip-list>
  <md-chip color="primary" selected="true">
    Selected - Primary
  </md-chip>
  <md-chip color="accent" selected="true">
    Selected - Accent
  </md-chip>
</md-chip-list>
Selected - Primary Selected - Accent
