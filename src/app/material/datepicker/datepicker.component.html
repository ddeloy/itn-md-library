Datepicker A material 2 component for datepicker

MdDatepickerModule - Official Documentation

Datepicker allows users to enter a date either through text input, or by choosing a date from the calendar.

A datepicker is composed of a text input [input element] and a calendar pop-up [md-datepicker element], connected via the mdDatepicker property on the text input. Datepicker is toggled via an icon button, with its [mdDatepickerToggle] property/attribute bound to the id of the md-datepicker element.

Touch UI mode - The datepicker normally opens as a popup under the input. However, this is not ideal for touch devices. Therefore, setting touchUi property to true on md-datepicker will enable a more touch friendly UI where the calendar opens in a large dialog.
