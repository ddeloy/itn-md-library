Tooltip Tooltips are text labels that appear when the user hovers over, focuses on, or touches an element.

MdTooltipModule - Official Documentation

Tooltip is attached to a host element by using mdTooltip directive.

Tooltip - Basic Example 
<button md-raised-button md-icon-button color="primary"
  mdTooltip="Print">
  <md-icon>search</md-icon>
</button>

Advanced Controls

Positioning - by default, tooltips appear below the host element. This, however, can be configured using mdTooltipPosition.

Show and Hide Delay - by default, tooltips appear and hide instantly. This, however, can be configured using mdTooltipShowDelay and mdTooltipHideDelay, respectively. Values provided to these attributes should be in milliseconds.

Tooltip Positioning and Delays 
<button md-raised-button color="primary"
  [mdTooltip]="message"
  [mdTooltipPosition]="position"
  [mdTooltipShowDelay]="showDelay"
  [mdTooltipHideDelay]="hideDelay">
  Mouse over to see the tooltip
</button>

Show and Hide

Tooltips can be programmatically shown or hidden using show and hide methods. They both accept a number in milliseconds to delay before applying the display change.

toggle method can be used as well to switch between the tooltip being shown or hidden.

Tooltip Methods 
<button md-raised-button md-icon-button color="primary"
  #tooltip="mdTooltip"
  mdTooltip="Add to Cart"
  mdTooltipPosition="above">
  <md-icon>shopping_cart</md-icon>
</button>

<p>
  <button md-raised-button color="primary" (mouseenter)="tooltip.show()">
    Show tooltip
  </button>
  <button md-raised-button color="primary" (mouseenter)="tooltip.hide(0)">
    Hide tooltip
  </button>
  <button md-raised-button color="primary" (mouseenter)="tooltip.toggle()">
    Toggle tooltip
  </button>
</p>
