Slider Panel <md-sidenav> is a panel that can be placed next to or above some primary content. A sidenav is typically used for navigation, but can contain any content. Slider Panel - with Menu Documentation

Added as a custom component SideMenuItem to the Library

So in whatever component you want to use slider panel with menu - do the following:


import { SidenavMenuItem } from '@ddeloy/dd-library';

Please see below example for code details.
Slider Panel - with Menu 
<md-toolbar color="primary"></md-toolbar>
<md-sidenav-container class="sidenav-container" style="height: 500px;">
  <md-sidenav class="sidenav-menu" mode="side" opened align="start">
    <ul>
      <sidenav-item *ngFor="let menuItem of menuItems" [menuItem]="menuItem"></sidenav-item>
    </ul>
  </md-sidenav>
  <div class="sidenav-content">
    <strong>Slider Panel Content</strong>
    <p>Rest of the content goes here...</p>
  </div>
</md-sidenav-container>
