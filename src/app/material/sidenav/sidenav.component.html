Slider Panel <md-sidenav> is a panel that can be placed next to or above some primary content. A sidenav is typically used for navigation, but can contain any content. Slider Panel - with Menu Documentation Create a class model: sidenav-menu-item.ts 
        /* I usually create my models in something like shared/models/ */
        export class SidenavMenuItem {
        public secTitle: string;
        public title: string;
        public link: string;
        public children: SidenavMenuItem[];
        public hasChildren: boolean = false;
        public pageTitle: string;

        constructor(secTitle: string, title: string, pageTitle: string,
        link: string, children: SidenavMenuItem[]){
        this.title = title;
        this.secTitle = secTitle;
        this.link = link;
        this.children = children;
        this.hasChildren = (children!=null && children.length!=0);
        this.pageTitle = pageTitle;
          }
        }
Create a new component named sidenav-item
With @angular/cli: ng g c sidenav-item
This will create a new directory with ts, html, css and test file
It will automatically be imported into app.module.ts for you
sidenav-item-component.ts 
        import { Component, Input } from '@angular/core';
        import { SidenavMenuItem } from '../sidenav-menu-item';
        /* Import the model you created in the above step
        The path will vary based on where you decide to create the model
        I usually create my models in something like shared/models/ */

        @Component({
        selector : 'sidenav-item',
        templateUrl: './sidenav-item.component.html'
        })

        export class SidenavItemComponent {
        @Input() menuItem: SidenavMenuItem;
        private _showChildren: boolean = false;

        clicked(event: MouseEvent) {
        this._showChildren = !this._showChildren;
        }
        }

Then the markup to create the example below would look like this: 
        <md-toolbar color="primary"></md-toolbar>
        <md-sidenav-container class="sidenav-container">
          <md-sidenav class="sidenav-menu" mode="side" opened align="start">
            <ul>
              <sidenav-item *ngFor="let menuItem of menuItems" [menuItem]="menuItem"></sidenav-item>
            </ul>
          </md-sidenav>
          <div class="sidenav-content">
            Content goes here
          </div>
        </md-sidenav-container>
